The Metric
HOW ICR IS CALCULATED

ICR = mean(R_t+1) across all actions in a run. Every action is stored as a CIEU five-tuple. No LLM involved. Fully deterministic. k9log verify-log confirms nothing was tampered.

Xt
Context
What the agent saw before acting
["CONSTRAINTS.md", "auth.py"]
Ut
Action
What the agent actually did
write_file("protected.py")
Y*t
Intent
The outcome the agent expected
"add auth module"
Yt+1
Outcome
What actually happened
BLOCKED — violation
Rt+1
Score
Conformance: 0.0–1.0
0.0 / non-conformant
