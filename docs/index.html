Two ways to use K9

Live monitoring or post-hoc audit

Hook K9 in before your agent runs, or point it at an existing codebase to audit what already happened.

⚡

Live Agent Monitoring

Hook K9 into your agent before it runs. Every action is recorded in real time. Violations are flagged the moment they happen — not after the damage is done.

Real-time · Zero disruption
Claude Code · Cursor · LangChain · AutoGen · OpenClaw · Any Python
🔍

Post-hoc Codebase Audit

Already have a codebase written by AI? Point K9 at it and run a structural audit — detect violations, dead imports, scope breaches without running anything.

Static analysis · No execution needed
Input: repo path · Output: causal violation report + remediation commands
Integration Wizard

1

Install K9 Audit

pip install k9audit-hook
2

Drop the hook into your project

Zero code changes — K9 hooks into Claude Code's event stream automatically via settings.json.

// .claude/settings.json { "hooks": { "PostToolUse": [{ "matcher": ".*", "hooks": [{ "type": "command", "command": "python -m k9log.hook" }] }] } }
3

Add CONSTRAINTS.md to your project

Claude Code reads this before acting. K9 enforces it after. Use the Rule Builder to generate it automatically.

# CONSTRAINTS.md ## Rules - NEVER write to production configs - NEVER use staging.internal URLs - Only modify files under ./src/**
4

Run and verify

k9log stats # see recorded events k9log trace --last # inspect last deviation k9log verify-log # confirm chain intact
✓

Trigger your first violation (30 seconds)

Run this in your project — K9 will catch it immediately:

python -c " from k9log.core import k9 @k9(deny_content=['staging.internal']) def write_config(content): return content write_config('https://api.staging.internal/v2') "

Then run:

k9log trace --last # see the violation with full causal chain

✓ If you see "VIOLATION — deny_content" you're fully set up.

Already have AI-written code? Audit it now.

    
Point K9 at any existing codebase — even one written entirely by AI without K9 running. The static analyzer detects violations without executing anything.
