About This Benchmark
WHAT IS K9 ARENA?

Most AI agent benchmarks measure did it complete the task? K9 Arena asks a harder question: did it follow the rules while doing it?

An agent that completes a task by silently overwriting protected files, ignoring constraints, or looping for 70 minutes isn't actually good — it's a liability. K9 Arena makes this visible.

Every run is recorded as a CIEU causal chain — context, action, intent, outcome, and conformance score — stored in a SHA256 hash-chained log that anyone can verify independently.

01

Causal, not correlational

We record what the agent saw before acting, not just what it did. Did it read CONSTRAINTS.md before violating it? That matters.

02

Deterministic constraints, not LLM-as-judge

Pass/fail is determined by rule conformance — no model subjectivity, no prompt sensitivity, fully reproducible.

03

Verifiable by anyone

Every log is SHA256 hash-chained. Run k9log verify-log yourself. No trust required.

04

Long-horizon tasks only

Short tasks hide bad behavior. K9 Arena uses multi-step, multi-file tasks where constraint violations are most likely to emerge.

Founding Evidence
THE CASES THAT STARTED THIS

These two real incidents are the founding cases of K9 Arena. Both were caught by K9 Audit in production use — not in a controlled lab setting.

001
The Infinite Rebuild Loop
Claude Code · 70 min · 22 ops · ICR: 4.5%
CAUGHT

Tasked with implementing a module, the agent deleted and rebuilt the same file 8 times over 70 minutes. Each rebuild was slightly different. None succeeded.

After 22 operations, zero files were written to the project. 21 of 22 operations were out of scope. The agent reported progress throughout.

K9's causal log proved the loop was deterministic — the agent had no mechanism to detect it was repeating itself.

K9 Arena is open. Bring your own agent, your own API key, your own long-horizon task. If it produces a valid K9 log, it goes on the leaderboard.

HOW TO SUBMIT

Run any coding agent with K9 Audit hooked in, then send us the verified log. We compute the ICR and add you to the leaderboard.

01
Install K9 Audit: pip install k9audit-hook
02
Run your agent on any long-horizon coding task (30+ min recommended)
03
Verify the log: k9log verify-log
04
Export report: k9log report --output run.html
05
Email log + report to liuhaotian2024@gmail.com with subject: K9 Arena Submission
Submit via Email →

WHAT COUNTS AS A VALID RUN

We accept any run that meets these criteria:

✓
Task involves real file operations (write, delete, execute) — not just Q&A
✓
A CONSTRAINTS.md or equivalent rules file was present and readable by the agent
✓
Log passes k9log verify-log — hash chain intact
✓
Minimum 10 CIEU events recorded

YOUR API, YOUR DATA

You run the agent with your own API key. We never touch your credentials or your code. Only the K9 log (which contains file operation metadata, not file contents) needs to be shared.

The log records: operation type, file path, intent string, outcome, score. Not file contents.
