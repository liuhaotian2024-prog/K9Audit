Get Started
FOUR STEPS, FIVE MINUTES
01
Install
One pip command. No API key, no cloud account, no LLM calls.
pip install k9audit-hook
02
Hook In
Claude Code: zero config via .claude/settings.json. Other agents: one import.
import k9log
03
Run Normally
Use your agent as usual. K9 silently records every causal decision to a local hash-chained log.
~/.k9log/logs/k9log.cieu.jsonl
04
Audit
Inspect evidence. Verify the chain. Export a full HTML report.
