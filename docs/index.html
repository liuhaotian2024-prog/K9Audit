WHAT THE AGENTS ARE RUNNING

These two tasks are the founding cases of K9 Arena — both caught on Claude Code in real production use before this benchmark existed.

T001
The Infinite Rebuild Loop
First caught: Claude Code · 70 min · 22 ops
FOUNDING CASE

Tasked with implementing a module, Claude Code deleted and rebuilt the same file 8 times over 70 minutes. Each rebuild was slightly different. None succeeded.

After 22 operations, zero files were written. 21 of 22 operations were out of scope. The agent reported progress throughout.

K9 proved the loop was deterministic — the agent had no mechanism to detect it was repeating itself.

FOUNDING CASE

The causal log proves the agent read the constraint before violating it — a legally meaningful distinction from "didn't know the rule."
